# P2P Chat Signaling Server

Минимальный сигнальный сервер на базе PeerJS для установки WebRTC соединений между устройствами.

## Описание

Этот сервер используется только для начальной установки P2P соединения между двумя устройствами. После установки соединения через WebRTC DataChannel, сервер не участвует в передаче данных.

## Функции

- Генерация 6-значных числовых кодов для подключения
- Обмен SDP и ICE кандидатами между устройствами
- Мониторинг активных соединений
- CORS поддержка для мобильных приложений

## Установка

```bash
npm install
```

## Запуск локально

```bash
npm run dev
```

## Деплой на Render

1. Создайте новый Web Service на [Render](https://render.com)
2. Подключите GitHub репозиторий
3. Укажите настройки:
   - **Build Command:** `npm install`
   - **Start Command:** `npm start`
   - **Environment:** Node
4. Добавьте переменные окружения:
   - `PORT`: 3000
   - `PEER_PORT`: 9000
   - `NODE_ENV`: production
5. Нажмите "Create Web Service"

## API Endpoints

- `GET /` - Health check
- `GET /status` - Статус сервера и количество подключений
- WebSocket `/` - PeerJS signaling endpoint

## Безопасность

- Сервер не хранит сообщения или файлы
- Все данные передаются напрямую между устройствами через WebRTC
- WebRTC использует встроенное DTLS/SRTP шифрование
